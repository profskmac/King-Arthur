# ♛ The Legend of King Arthur — Royal Webquest

An upscale educational webquest site about King Arthur, the Knights of the Round Table, the Holy Grail, and Medieval life.

## Files

| File | Description |
|---|---|
| `index.html` | Entry point — redirects to the home page |
| `king-arthur-webquest.html` | Home page with intro, process, roles, and image gallery |
| `investigator.html` | Role 1: King Arthur, Excalibur & Arthurian Legends |
| `adventurer.html` | Role 2: Knights of the Round Table & Chivalry |
| `traveler.html` | Role 3: The Holy Grail |
| `teacher.html` | Role 4: Life in the Middle Ages |

## How to Publish

### Option 1 — Netlify Drop (Easiest, Free)
1. Go to **[netlify.com/drop](https://netlify.com/drop)**
2. Drag the entire unzipped folder onto the page
3. Your site goes live instantly at a `*.netlify.app` URL

### Option 2 — GitHub Pages (Free)
1. Create a new GitHub repository
2. Upload all files to the repository root
3. Go to **Settings → Pages → Source: main branch / root**
4. Your site is live at `https://yourusername.github.io/repo-name/`

### Option 3 — Any Web Host (cPanel, etc.)
1. Upload all `.html` files to your `public_html` folder via FTP or File Manager
2. The site will be accessible at your domain root

### Option 4 — Google Sites / Local Use
Open `king-arthur-webquest.html` directly in any browser. All links work locally.

## Teacher Edit Mode

The site includes a hidden edit mode for customizing all text content:

- **Desktop**: Press `Ctrl+E` (or `Cmd+E` on Mac) to reveal the Edit toolbar
- **Mobile**: Triple-tap the bottom-right corner of the screen
- Click any text element to edit it in place
- Use the floating toolbar for Bold / Italic / Underline formatting
- Press **Save** to preserve your changes for the session
- Press **Reset** to restore original content

> **Note:** Edits are session-only (in-memory). To make permanent changes, edit the HTML files directly.
